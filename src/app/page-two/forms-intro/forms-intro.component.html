Forms Intro

In this module i have placed two forms one is made using the ngModel directive on Form input fields and the ngForm on the Form element itself.
And the second one, was made using reactive forms module, where we used FormGroup, FormControl and FormArray constructors to make form objects and then in the template mark the corresponding fields with [formGroupName] etc
The reactive forms, also submit to a firebase RT DB, and fetch results using httpClient Module of Angular.
