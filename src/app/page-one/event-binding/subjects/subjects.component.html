The input field has a keyup event that triggers an event handler, and passes the value of the field using template reference and the handler then uses the NEXT method on the subject which was initialized in the service. so every time user has a keyup the value is sent to handler which emits a value and the parent component has a subscription setup in it ngOnInit method that listens to any future data sent.

And inside the subscription callback we set the data received to the local prop of it.
